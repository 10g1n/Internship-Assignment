# Internship-Assignment

## How to build and run the program

### 1. Prerequisites
    • JDK Version: Java 8 or higher (I used 21.0.2)
	• Operating System: MacOS
	• IDE: You can use IntelliJ IDEA, Eclipse, VSCode, or just the command line.

### 2. Clone the Repository
    git clone https://github.com/your-username/GroupAnagrams.git 
    cd GroupAnagrams
#### 2.1 Compile the Program
    javac GroupAnagrams.java
#### 2.2 Run the Program
    java GroupAnagrams
